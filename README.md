Sakurairo🌸: 基于Sakura的分支主题
===

基于主题 [Sakura](https://github.com/mashirozx/Sakura) 的基础上修改的Sakurairo主题

![Sakurairo](https://asuhe.jp/wp-content/uploads/2020/03/screenshot.jpg)

相比原主题，它有这些大变化：

-全局主题色改成粉色（#EE9CA7）。

-移动端全屏，并显示社交卡片。

-添加花瓣飘落JS

-友链页界面优化

相比原主题，它默认开启：

-调用本地CSS

-社交栏样式为独立成行

-开启PJAX

主题使用说明见：<https://2heng.xin/theme-sakura/>
